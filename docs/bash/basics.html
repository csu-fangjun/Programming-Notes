Get the current directory of the script¶

cur_dir=$(cd $(dirname $BASH_SOURCE) && pwd)
echo $cur_dir

date¶

date "+%F %T"
# 2010-01-02 08:01:02

array¶

a=(
hello
world
)

for i in ${a[@]}; do
  echo $i
done

Output

hello
world

Block Comment¶

Refer to Block Comments in a Shell Script.

:<<'EOF'
hello'world
foo
bar
EOF

Hint

: can be omitted. But it is not portable if : is missing.
