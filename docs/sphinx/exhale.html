Exhale¶

Exhale can be used to generate documention for C++ code which is documented with Doxygen.

The advantage of using Exahle is that it can generate the sphinx style webpage pages which looks more modern than Doxygen.

Install Exhale¶

pip install exhale

At the end, it prints some message similar to the following:

Installing collected packages: lxml, soupsieve, beautifulsoup4, bs4, breathe, exhale
Successfully installed beautifulsoup4-4.8.2 breathe-4.14.1 bs4-0.0.1 exhale-0.2.3 lxml-4.5.0 soupsieve-2.0
