Useful Extensions¶

sphinx.ext.extlinks¶

If there are lots of URLs linking to the same site, then this extension can be helpful.

sphinx.ext.napoleon¶

Use it to convert Google docstring style to a format supported by sphinx.

Here is an example of Google style docstring: https://www.sphinx-doc.org/en/master/usage/extensions/example_google.html#example-google
