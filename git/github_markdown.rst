

GitHub Markdown
===============

**Collpase**

  .. code-block::

    <details>
      <summary> thanks </summary>

      ```cpp
      void thank_you() {}
      ```

    </details>

  Note that there MUST be a blank line after ``<summary>``.

  Refer to `<https://gist.github.com/pierrejoubert73/902cc94d79424356a8d20be2b382e1ab>`_.

  Use ``<details open>`` to leave it open by default.
