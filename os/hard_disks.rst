
Hard Disks
==========

- spindle
- arm, head
- disk == platter.
- track
- cylinder
- sector: unit for read/write
- example: hard disk sector is 4KB, while OS block is 8KB
- CHS: cynliner, head, sector
- LBA: logical block addressing
